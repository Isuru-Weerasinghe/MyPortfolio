Portfolio Details

Sri Lanka Railways

The Railway Management System (RMS) is an innovative web application developed using PHP for both frontend and backend, with MySQL as the database management system. Created as part of the "Project II" course module during the second year, this system aims to modernize railway operations in Sri Lanka by automating key functions and addressing the limitations of traditional management methods. RMS provides a range of user-centric features, including real-time train schedules, secure online ticket reservations, an e-catering service, and a chatbot for customer support. It also includes advanced functionalities such as QR code-based ticket verification, payment gateways for secure transactions, and real-time notifications about train delays.

The system emphasizes scalability, security, and efficiency, offering multi-role access for administrators, railway operators, and station staff to manage operations effectively. Designed with a user-friendly interface, RMS ensures seamless accessibility across devices. Its robust database handling with MySQL enables efficient data storage and retrieval, while PHP ensures dynamic and responsive functionality. Future enhancements include a dedicated mobile app and personalized user recommendations, showcasing the project's potential as a comprehensive and cutting-edge transportation management solution.
